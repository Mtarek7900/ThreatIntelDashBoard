# ThreatIntelDashBoard
## Threat Intelligence Dashboard — Backend API

A secure backend service that aggregates and analyzes threat intelligence data from multiple sources and exposes it through clean REST APIs. Built with Java + Spring Boot, it allows users to submit IPs, domains, or file hashes and receive structured risk assessments, history tracking, and normalized threat reports.

## Why This Project Matters

This project showcases:

- Strong Java backend engineering
- Security mindset & real-world SOC understanding
- API design, integrations, logging, and resilience
- Secure development practices
- Perfect for security engineering, backend, and DevSecOps roles.

## Features

- Submit IP / Domain / Hash for threat analysis
- Queries external threat-intel APIs (ex: URLHaus, VirusTotal, etc.)
- Normalizes and scores threat results
- Stores lookup history in database
- Secure API structure & validation
- Designed for scalability & production-readiness
